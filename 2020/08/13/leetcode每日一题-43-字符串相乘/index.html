43. 字符串相乘

难度 中等

给定两个以字符串形式表示的非负整数 num1 和 num2，返回 num1 和 num2 的乘积，它们的乘积也表示为字符串形式。

示例 2:

说明：

num1 和 num2 的长度小于110。
num1 和 num2 只包含数字 0-9。
num1 和 num2 均不以零开头，除非是数字 0 本身。
不能使用任何标准库的大数类型（比如 BigInteger）或直接将输入转换为整数来处理。

思路

创建dp数组模拟乘法运算。

const dp = Array(len1+len2).fill(0);

从后向前遍历两个字符串，对当前的两个字符做乘法。

将进位存储在前一位，将个位数的值存储在当前位。

dp[i+j+1] = t % 10;

dp[i+j] = t / 10 | 0;

对于下一次迭代，继承上一次迭代的进位状态。

t = n1 * n2 + dp[i+j+1]

遍历完毕，dp数组已被填充，将不必要的前导零shift出去。

while(dp[0]===0){ dp.shift(); }

最后，返回dp数组的join。

注意dp数组为空的特殊情况，这意味这dp数组之前全是0，所以我们在返回的时候要对dp数组的长度做判断，如果dp数组为空，直接返回“0”。
